Simulator
=========


Process Scheduler.



Version
----

1.0

Requirements
--------------


* Gcc
* Pthread


How to use
--------------

```sh
git clone https://nairitya@bitbucket.org/nairitya/simulator.git
cd simulator

update process.txt
    write the process name, arrival time. burst time and priority.

run 'gcc -pthread -o simulator simulator.c'
Now run the executable as './simulator'
    
```
